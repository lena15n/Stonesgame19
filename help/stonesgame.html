Назначение системы

Данное приложение представляет собой программу, реализующую автоматизированную систему составления и разгадывания классического кроссворда.

Приложение предназначено для составления кроссворда с использованием собственного словаря понятий либо автоматической его генерации. Пользователь может производить разгадывание кроссворда и поиск ошибок.

Условия работы системы

Для корректной работы системы необходимо наличие соответствующих программных и аппаратных средств.

Требования:

  • персональный компьютер с ОС Windows XP или выше;

  • не менее 1425 Мб оперативной памяти;

  • 10,4322 Гб на жестком диске;

  • дисплей с разрешением не менее 1024 x 768 пикселей;

  • клавиатура или иное устройство ввода;

  • мышь или иное манипулирующее устройство.

Работа в режиме «Пользователь»

При выборе опции главного меню «Разгадывание» происходит переключение системы на роль «Пользователь». Откроется окно с пустыми областями просмотра кроссворда и словаря. Для начала необходимо создать кроссворд автоматически или загрузить существующий кроссворд.

Автоматическое составление кроссворда

Создать кроссворд автоматически можно при выборе в выпадающем списке «Кроссворд» пункта меню «Сгенерировать». При этом откроется окно настройки параметров будущего кроссворда (рисунок 2). В нем указаны три поля для заполнения:

  • путь к словарю;

  • длина сетки кроссворда;

  • ширина сетки кроссворда;

Поле «Словарь» обязательно для заполнения, для этого стоит нажать на кнопку «Обзор...» и выбрать необходимый словарь. Остальные поля задаются с помощью кнопок увеличения значения на единицу и уменьшения значения на единицу. Если оставить их неизменными, заполняются значениями по умолчанию.

Загрузка существующего кроссворда

Чтобы загрузить существующий кроссворд, необходимо выбрать в выпадающем списке «Кроссворд» пункт меню «Загрузить». Будет открыто окно, где нужно указать имя необходимого кроссворда. Кроссворд будет открыт.

Разгадывание кроссворда

После того как кроссворд был выбран, можно приступить к его разгадыванию. В наличии у пользователя есть два типа подсказок (рисунок 3):

  • открыть букву;

  • открыть все слово целиком.

Для выбора подсказки достаточно выделить необходимую букву слова (слово) и нажать на кнопку «Открыть букву» («Открыть слово»).

По завершению игры следует нажать кнопку «Завершить игру». Система выдаст результат и откроет ответы.

Работа с понятиями словаря

При работе с понятиями словаря доступны следующие функции:

  • добавление;

  • редактирование;

  • удаление.

При добавлении или редактировании слова в словаре доступны для редактирования поля: понятие и определение (рисунок 7).

Поиск понятий по маске

Определенные понятия находить просто при использовании поиска по маске ? набора букв и специальных символов: «?» (обозначает один символ) и «*» (обозначает любую последовательность символов). После ввода маски в поле следует нажать на кнопку поиска понятий по этой маске. В словаре останутся только слова, соответствующие ей. Далее маску можно очистить.
